Welcome to my GIS Portfolio.

GIScience is a scientific study that applies different Geographical Information System (GIS) to research, analyze and understand various spatial questions, and eventually create solutions. GIS has a plethora of applications, ranging from using Google Maps to find your way to work, to working with huge governmental data sets to determine the location of a new highway. GIS is unique in that it can offer so many different avenues of analysis and data representation for different geospatial queries.

The goal of this course was to offer an introduction into both GIScience as a field of study, and gain familiarity with different types of GIS software, especially QGIS and ArcGIS. We had various different assignments ranging from vector maps using isochrone layers to vegetation indices maps using spectral signature satellite data. This course facilitated the development of not only my cartography skills, but likwise develop my coding, website design and analysis skills. In total, there were six assignments throughout the course, please feel free to look around at the different pages.

Choropleth Map

A choropleth map depicting Goal 5: Gender Equality, of the UN's Sustainable Development Goals.

Relative Elevation Map (REM)

A relative elevation map of Holland's Diep.

Vegetation Indices Map

Various maps created using Google Earth Engine, showing the results of deforestation using vegetation indices, near Lichinga, Mozambique.

Vector Map

A vector map showing walking distances to bus stops in Waltham.

Land Use Classification

Land use classification map based on satellite imagery of Utrecht.

Multi-Criteria Analysis

A discussion about a MCA research paper on flooding risk in the Ghatal Block.
